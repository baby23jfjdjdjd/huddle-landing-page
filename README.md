# Frontend Mentor - Huddle Landing Page Solution

This is my solution to the **Huddle Landing Page with a Single Introductory Section** challenge on Frontend Mentor. The goal of this project was to build a responsive landing page that closely matches the provided design while following modern HTML and CSS best practices.

## Overview

This project is a fully responsive landing page built with semantic HTML5 and CSS3. It adapts smoothly to different screen sizes using Flexbox and media queries.

## What I Learned

While building this project, I improved my understanding of:

* Creating responsive layouts with Flexbox
* Using media queries to optimize layouts for different screen sizes
* The difference between `width` and `max-width`
* Writing more semantic HTML
* Improving accessibility with descriptive `alt` attributes
* Creating hover effects that only apply on devices that support hovering using:

```css
@media (hover: hover) {
  .btn:hover {
    /* Hover styles */
  }
}
```

I also learned how to publish a project using **GitHub Pages** and manage my code with **Git** and **GitHub**.
